Product Automatic Sale Price

Calcula el precio de venta (list_price) a partir del costo usando fórmulas heredables de producto o categoría, con acción masiva para recalcular.

  • Fórmulas: porcentaje, coeficiente o suma fija.
  • Fallback producto → categoría.
  • Acción masiva en lista de productos.
  • Icono en static/description/icon.png.
